TO run the attack, do the following:

	press "make" in the terminal in the directory of all the files without quotes.

be sure to be connected to the iitk network.


Also, make sure the internet is very fast. We have set the timeout in expect script to 5 seconds.
If your internet is very slow, make sure to change the timeout accordingly in the expect script named "inputs1.exp"

The attack will take around 10 to 20 minutes to complete on normal internet speed. 
ON fast internet speed, it will take atmost 3 minutes.
